# Simple Wedding Invitation Landing Page 

**Note:** This project has been refactored from Indonesian to English. All text, class names, and file names have been updated to use English terminology.

# Sections/Features
- Main Info
- Countdown to D-Day
- Time and Place Info
- Add to Calendar Button (Google Calendar)
- Map Direction Button (Google Map)
- Send Message Button (WhatsApp API)
- Photo Gallery
- RSVP/Attendance Confirmation

# Stack
- Netlify (https://netlify.com)
- Bulma CSS (https://bulma.io)

# Key Changes in English Refactoring
- **CSS Files**: `menikah.css` → `wedding.css`
- **JavaScript Files**: `menikah.js` → `wedding.js`
- **Class Names**: 
  - `nama-lengkap` → `full-name`
  - `waktu` → `time`
  - `tempat` → `venue`
  - `foto1-6` → `photo1-6`
  - `tempatwaktu` → `datetime-location`
- **Section IDs**: 
  - `#Waktu` → `#time`
  - `#lokasi` → `#location`
  - `#tentang-pasangan` → `#about-couple`
- **Countdown Labels**: Days, Hours, Minutes, Seconds (previously in Indonesian)
